How to use

[1] Select the type of graph you would like to make from the dropdown.

  • If histogram, input will pop up to define the amount of categories in x-axis you want defined.

[2] After choosing from the dropdown, the instructions for CSV file format will show below

    Bar Graph

  1. Title of the graph
  2. x-axis label, y-axis label
  3. Then the rest of the rows should be the x-value (words/ categories), y-value

    Normal Graph

  1. Title of the graph
  2. x-axis label, y-axis label
  3. Then the rest of the rows should be the singular values in a column.

    Scatterplot

  1. Title of the graph
  2. x-axis label, y-axis label
  3. Then the rest of the rows should be the x-value, y-value

    Histogram

  1. Title of the graph
  2. x-axis label, y-axis label
  3. Then the rest of the rows should be the singular values in a column.

    BoxPlot

  1. Title of the graph
  2. x-axis label, y-axis label
  3. Then the rest of the rows should be the singular values in a column.

[3] Then use the file selector, input the file you made and let the program do the work.

[4] When the graph is displayed on the screen, hover over the bars and it will display the value of that bar.
