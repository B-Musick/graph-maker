How to use

- Create a CSV file with the data you have. CSV file should be formatted in the following way

  1. Title of the graph
  2. x-axis label, y-axis label
  3. Then the rest of the rows should be the x-value, y-value

- Select the graph type you want to make

  • If histogram, input will pop up to define the amount of categories in x-axis you want defined.

- Then use the file selector, input the file you made and let the program do the work.

- When the graph is displayed on the screen, hover over the bars and it will display the value of that bar.
